# HabitLink Personal

Личная PWA-версия трекера привычек. Работает без сервера, хранит данные в `localStorage`, поддерживает офлайн-режим через service worker и может быть опубликована на GitHub Pages.

## Запуск

Откройте `index.html` локально или раздайте папку любым статическим сервером.

```powershell
python -m http.server 4173
```

После публикации на HTTPS откройте сайт на iPhone в Safari и выберите «Поделиться» -> «На экран Домой».

## GitHub Pages

Опубликуйте содержимое этой папки в репозиторий GitHub и включите Pages:

1. Settings -> Pages.
2. Source: `Deploy from a branch`.
3. Branch: `main`, folder: `/root`.

После открытия опубликованного HTTPS-адреса service worker закеширует приложение, и оно продолжит открываться офлайн.

## Что уже есть

- создание и редактирование привычек;
- расписание по дням недели;
- отметка выполнения за сегодня;
- серии дней и достижения;
- лента личной активности;
- профиль, настройки, уведомления-мокап;
- экспорт и импорт данных JSON;
- манифест PWA и офлайн-кеш.
